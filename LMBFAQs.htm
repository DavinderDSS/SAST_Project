LMB FAQs

 

  • What if ‘Labour Planning’ page not displayed?

Soln:  Either no material for planning is being selected or material selected for planning may not be in the norms list of that plant. In a list of selected material for planning there should be at least one material from the norms list.

 

  • What if the approved plan needs to be changed?

Soln:  This situation may occur in case some high priority job is to be done. In this case supervisor can go to ‘Direct Execution’ page for planned or unplanned activity. This allows the supervisor to execute the task(s) which was not planned for him. But the supervisor also has to execute the planned activity for him with zero values as this will not cause any problem and at the time of approve execution system will not show that supervisor’s execution is pending.

 

  • What values to be put for labour required fields in ‘Planning’ page in case planning for supervisor is not done for the full shift?

Soln: Put the value in duties. For example, if the task to the supervisor is assigned for 4 hours with 5 labours then it is 5 X 4 = 20 labour hours, means 2.5 duties (calculated by considering 1 duty = 8 hrs).

 

  • What values to be put for labour fields in ‘Over Time’ (OT) page?

Soln: Put the value in duties. For example, if the over time is done for 4 hours with 5 labours then it is 5 X 4 = 20 labour hours, means 2.5 duties (calculated by considering 1 duty = 8 hrs).

Note: On approve execution page OT value displayed is the total contract labours OT value, here company labours OT value is not considered in order to compare OT duties of contractor only.

 

  • What value to be put for labour count field in ‘Process Loss’ page?

Soln:  Total number of labours idles during the period.

 

  • What if one add rows in any of ‘Labour Planning’ page, ‘OT’ page and ‘Process Loss’ page and do not enter information in all rows?

Soln: System will accept only those rows in which minimum required information is entered and rest of the rows will be ignored.

 

  • While running contractor annexure report – “Rates difference is found in selected date range. Please select proper dates” message is displayed.

Soln: This is only because either of your billing rates is changed in between selected date range. For example, you are trying to get contractor annexure for the month of March and you select the dates from 1st march to 31st march, and either of your billing rates (labour rate, pf charge, service charge, service tax, tds, esi etc.) is changed from 16th march onwards. In this case, you have to get two different bills for the month of March i.e. one from 1st march to 15th march and other from 16th march to 31st march.
